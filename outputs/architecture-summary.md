## Data Flow (startup routine)

```
main()
  → PatientService.getAllPatients("system-startup")
      → PatientRepository.findAll()          // JDBC SELECT * FROM patients
          → DatabaseConfig.{DB_URL,DB_USER,DB_PASSWORD}
      → AuditLogger.logAccess(patient, user) // for every patient returned
          → Patient.toString()               // logs full PII to stdout
  → PatientService.sendAllReminders()
      → PatientRepository.findAll()          // second full table scan
      → NotificationService.sendReminder(patient)
          → DatabaseConfig.NOTIFICATION_API_KEY
          → Patient.getFullName()
```
